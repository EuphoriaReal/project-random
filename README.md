# Projet pédagogique – Étude des vulnérabilités

## Description

Ce dépôt contient un projet **à but strictement pédagogique** visant à étudier, comprendre et expérimenter des vulnérabilités informatiques dans un cadre académique contrôlé.

**Ce projet est destiné uniquement à l’apprentissage et à la recherche défensive.
Toute utilisation malveillante est interdite.**
